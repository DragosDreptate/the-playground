Décisions de design (issue #597)

  • Cover 1:1 (carré) — cohérent avec CoverBlock et le design system. Overlay possible : badge Démo / rôle.
  • Explorer → vertical, 3 colonnes (pleine largeur). Mon Espace (~672px) → vertical, 2 colonnes. Nombres provisoires, à valider au rendu réel.
  • Mobile (< sm) inchangé en Phase 1 (format horizontal actuel conservé). Refonte éventuelle en Phase 2.
  • Contenu inchangé, on change l'aspect. On reprend fidèlement les champs déjà affichés par chaque carte ; le seul déplacement est l'encart « prochain événement », qui passe de la colonne droite (desktop) au corps de la carte.
  • Explorer (PublicCircleCard) : cover (+ Démo) → catégorie → nom → description → ville → compteur d'événements à venir → avatars + membres → badge rôle (overlay) → prochain événement.
  • Mon Espace (DashboardCircleCard) : cover → catégorie → nom → ville → avatars + membres → badge En attente → prochain événement. Pas de description, de compteur, de CTA ni de badge de rôle (comme aujourd'hui).
  • Périmètre strict : #597 reste iso #596. On change uniquement l'affichage des cartes (horizontal → vertical, liste → grille). Aucun ajout lié au sélecteur Participant/Organisateur : pas de CTA Organisateur, pas de badge de rôle sur le dashboard. #596 viendra après.
  • Composant unifié piloté par variantes (public / dashboard), remplaçant PublicCircleCard + DashboardCircleCard. Nom à trancher (conflit avec circle-card.tsx existant → ex. CommunityCard).
  • Section « À la une » supprimée (ExplorerFeatured) : elle n'apporte rien aujourd'hui, on la retire (on ne l'harmonise pas).
  • Chargement progressif (infinite scroll) sur l'onglet Communautés d'Explorer, en remplacement du « Load More ». Onglet Événements non concerné cette itération.
  • Effet hover unifié (toutes les cartes, événement + Communauté) : remplacer le highlight rose actuel (hover:border-primary/30 + transition-colors) par une élévation neutre — translateY(-2px) + ombre renforcée, comme dans ce mockup. Concerne public-circle-card, dashboard-circle-card, public-moment-card, dashboard-moment-card, moment-card.
