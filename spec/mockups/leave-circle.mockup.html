Notes de design

Placement desktop : bouton dans la colonne gauche sticky, sous les stats. Discret (outline destructive sm) — pas le CTA principal.

Placement mobile : bas de page, après la liste des membres, séparé par un border-top. Accessible sans être intrusif.

Mobile : bottom sheet plutôt que Dialog centré — pattern natif iOS/Android, plus confortable au pouce.
4
Message d'avertissement : rassurer explicitement que les inscriptions aux événements restent actives. Réduit l'hésitation.
