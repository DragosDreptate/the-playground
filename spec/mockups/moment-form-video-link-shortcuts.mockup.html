Notes de design

Comportement
  • Les boutons ouvrent un nouvel onglet vers la plateforme
  • L'organisateur crée le lien, le copie-colle dans le champ
  • La détection automatique lit le domaine de l'URL
  • L'icône plateforme s'affiche dans le champ si détecté
  • Le hint change : "sera joint à l'invitation calendrier"
Plateformes détectées
  • meet.google.com → Google Meet
  • zoom.us → Zoom
  • teams.microsoft.com → Teams
  • Autres URLs → icône neutre, hint générique
ICS — amélioration connexe
  • Actuellement : LOCATION:En ligne (bug)
  • Après fix : LOCATION:https://meet.google.com/…
  • Bonus : propriété CONFERENCE;FEATURE=VIDEO
  • → bouton "Rejoindre" natif dans Google Calendar
Implémentation
  • Modifier moment-form-location-row.tsx
  • Détecter le domaine via new URL(value).hostname
  • Aucune API externe, aucun OAuth supplémentaire
  • Fix ICS : generate-ics.ts + route /api/moments/[slug]/calendar
