Notes d'implémentation

  • Page Circle : bouton "Quitter la Communauté" affiché si showMemberBadge = true (PLAYER, non-HOST)
  • Dashboard : icône ⋮ (MoreVertical) affichée sur la card si memberRole === "PLAYER"
  • HOST : aucun bouton Quitter — ni sur la page Circle, ni sur la carte Dashboard
  • Modale : AlertDialog shadcn/ui — même composant pour les deux entrées
  • Ré-adhésion : via inscription à un événement (comportement actuel, automatique)
  • Inscriptions futures : à trancher — MVP = garder les inscriptions existantes
